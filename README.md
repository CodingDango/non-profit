# Hope Foundation
A sleek, high-impact landing page for non-profit organizations to drive engagement and transparency.

[**Live Demo Link**](https://hopefoundation-global.vercel.app)

## Features

* **Animated Impact Stats:** Interactive counters for global outreach metrics.
* **Sticky-Side Testimonials:** Narrative-driven layout that keeps context visible during scroll.
* **Donation Flow:** Custom modal preview with multi-platform payment support.
* **Responsive UI:** Fully optimized for mobile devices.

## Tech Stack

* **Framework:** Next.js
* **Styling:** Tailwind CSS + shadcn/ui
* **Icons:** Lucide React
* **Animations:** Framer Motion + React CountUp
* **Deployment:** Vercel
